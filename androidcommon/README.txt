This contains core functionality that is shared between ODK Tables and ODK Survey.

Functionality will include:

- database and content provider base classes
- utility classes for http interactions
- utility classes for sdcard actions
- file parsers and processing functions (e.g., read csv file, insert into db)
- server synchronization functionality

What it does not provide:

- Tasks
- Services
- Content Providers
